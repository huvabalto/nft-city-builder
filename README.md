# NFT City Builder

An asynchronous, NFT-native city simulation protocol built for **Base** and **Farcaster Frames**.

## Core Ideas
- ERC-1155 people as productive agents
- ERC-721 buildings as scarce infrastructure
- Off-chain simulation, on-chain enforcement
- No inflationary token, no promised yield
